# backend-Sistem-Booking-Ruangan-Fasilitas
Aplikasi web untuk mengelola peminjaman ruangan dan fasilitas seperti laboratorium, ruang rapat, dan proyektor agar tidak terjadi bentrok jadwal. Pengguna dapat mengecek ketersediaan dan mengajukan booking, lalu setiap pengajuan akan melalui proses approval admin sebelum dinyatakan valid dan dapat digunakan.
